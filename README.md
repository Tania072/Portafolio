# Catálogo Digital con Efecto Libro

Un catálogo interactivo con efecto de pasar páginas como un libro real.

## Características
- ✅ Efecto libro 3D responsivo
- ✅ Navegación por teclado y touch
- ✅ Diseño adaptable para móviles
- ✅ Modal para imágenes
- ✅ Integración con WhatsApp

## Tecnologías
- HTML5
- CSS3 (Grid, Flexbox, Transformaciones 3D)
- JavaScript Vanilla
